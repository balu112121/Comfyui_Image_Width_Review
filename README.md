# 南光AI宽限图像 (Comfyui_Image_Width_Review)

## 功能描述
严格遵守AIGC规范，科学规范进行创作！
自动识别人物胸部和私处区域，严格采用马赛克进行遮挡；并分别以**矩形**（胸部）和**圆形**（私处）马赛克进行遮挡，用于规避内容审核。


---

## 🔧 安装步骤（快速版）

1. 创建文件夹 `ComfyUI/custom_nodes/Comfyui_Image_Width_Review/`。
2. 将上述所有文件按结构放入对应位置。
3. 执行 `pip install -r requirements.txt`（确保在插件目录下）。
4. 完全重启 ComfyUI。

## ✅ 验证

- 节点出现在 **南光AI/图像** 分类下，名称为 **南光AI宽限图像**。
- 所有参数名称均为中文，包含水平偏移和垂直偏移。
- 胸部输出矩形马赛克，私处输出圆形马赛克。
